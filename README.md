Analytics site for [Dolomite Margin](https://dolomite.io).

Includes support for the [DolomiteMargin](https://github.com/dolomite-exchange/dolomite-margin) Protocol

### To Start Development

###### Installing dependencies
```bash
yarn
```

###### Running locally
```bash
yarn start
```
